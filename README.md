# Language Translation Tool

A Streamlit-based language translation application with:
- Auto language detection
- Multi-language translation
- Audio output
- Speech-to-text via audio upload

## Technologies Used
- Python
- Streamlit
- deep-translator
- gTTS
- SpeechRecognition

## How to Run
pip install -r requirements.txt
streamlit run app.py
